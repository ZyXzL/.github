# .github

This repository is the GitHub organization profile repo. GitHub treats a repo named `.github` under an organization specially: the file at `profile/README.md` is rendered publicly on the organization's landing page (github.com/<ORG_HANDLE>), visible to anyone who visits the org.

## Publishing

1. On GitHub, create a new repository named exactly `.github` under your organization (e.g. `github.com/<ORG_HANDLE>/.github`).
2. Push this directory's contents to that repo (the `profile/README.md` path must be preserved).
3. The org profile will appear automatically — no additional settings needed.
4. Edit `profile/README.md` to replace all `<PLACEHOLDER>` tokens with your actual org handle, project names, and links.
